BELAJAR ...args | rest parameter

Apa itu args?

  • Sebuah cara untuk menangkap semua argument yang dikirim ke function
  • args bisa diganti nama apapun, contohnya ...Numbers, ...value, ...Doraemon

Contoh implementasi dalam code:

        
            function contoh(...args) {
            console.log(args);
            }

            contoh(1, 2, 3);          //[1, 2, 3]
            contoh('a', 'b', 'c');    //['a', 'b', 'c']
